DEMO:https://blog.lwxyfer.com/ghost-kai-fa-ji-lu/

# USAGE

loading ways supported:

- ES6
- AMD
- CommonJS
- IIFE
- UMD

Use rollup.js to bundle. Modify rollup.config.js to config loaders.

```
rollup -c
```
